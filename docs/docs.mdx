## Installation

<InstallScript packages="@gfazioli/mantine-depth-select" />

After installation import package styles at the root of your application:

```tsx
import '@gfazioli/mantine-depth-select/styles.css';
```

You can import styles within a layer `@layer mantine-depth-select` by importing `@gfazioli/mantine-depth-select/styles.layer.css` file.

```tsx
import '@gfazioli/mantine-depth-select/styles.layer.css';
```

## Animations

Add visual feedback with built-in animations. The component supports multiple animation types:

- **pulse**: Smooth pulsing effect (default)
- **flash**: Quick flashing
- **breathe**: Slow breathing effect
- **blink**: Regular blinking
- **glow**: Glowing effect with increased intensity

<Demo data={demos.animations} />

## Labels

Add descriptive labels to your DepthSelect indicators with full customization support:

<Demo data={demos.labels} />

The `label` prop accepts any React node, allowing you to use custom components like badges, icons, or formatted text. Use `labelPosition` to control whether the label appears on the left or right side of the DepthSelect.

## Styles API

`DepthSelect` supports [Styles API](https://mantine.dev/styles/styles-api/), you can add styles to any inner element of the component with `classNames` prop. Follow [Styles API](https://mantine.dev/styles/styles-api/) documentation to learn more.

You can customize the appearance of the `DepthSelect` component using the Styles API. This allows you to modify styles for various parts of the component, such as the root, depthSelect, and label.
