## 🧩 What this is

mantine-depth-select is a stack-style select component for React apps that use Mantine. It shows cards in a 3D depth view, so users can move through items with a clear sense of position.

It is built for interfaces where a plain dropdown feels too flat. The stacked view helps people scan options faster and keep track of where they are in the list.

## 🧠 How it works

The component shows a stack of cards instead of a flat list. The front card is the main choice. Cards behind it fade into the background with depth. When the user moves through the stack, the view shifts with a smooth transition.

This helps in places where users want to browse options without losing context. It gives the interface a more guided feel than a basic select box.
